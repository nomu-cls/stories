## 0. 飯村さんから学んだ構造

飯村さんの説明動画の構造を分析すると、こういう流れ：

```
① 共感（あなたも悩んでるよね）
② 常識破壊（犬だけ見てもダメなんです）
③ 解決策の提示（飼い主軸を育てる）
④ 証拠（受講生の変化事例：イライラ解消、犬の体臭改善、肝臓数値正常化…）
⑤ 自己紹介（実績・権威性・メディア）
⑥ 3ステップ（「私を生きる」→ カウンセリング実績 → プロ活動）
⑦ 講座の案内（審査制で希少性）
```

**きいちゃんが真似すべき点**:
- 「飼い主自身が変わらなきゃ」というテーゼ → **同じ**
- 受講生の変化事例 → **きいちゃんはまだない。だから今は「自分のストーリー」と「科学的根拠」で代替**
- 審査制・希少性 → **ゼロ期は5名限定で同じ構造が使える**

**きいちゃんが差別化すべき点**:
- 飯村さんは「カウンセリング技術」が武器 → きいちゃんは**「分子栄養学 × コルチゾール研究」が武器**
- 飯村さんは感覚・心理寄り → きいちゃんは**科学的根拠で裏付けできる**
- 飯村さんはフードの話はしない → きいちゃんは**栄養 + マインドの両方**ができる

## 2. 「5つの心の栄養素」— きいちゃん独自のフレームワーク

飯村さんは「飼い主軸」という**1つのコンセプト**。
きいちゃんは「5つの心の栄養素」という**フレームワーク**で差別化。

なぜ強いか → **栄養士だから「栄養素」で語れる**。これは飯村さんには絶対できない。

| # | 心の栄養素 | アイコン | 何を見ているか | きいちゃんの専門性 |
|---|-----------|---------|--------------|-----------------|
| 1 | **安心ごはん** | 🍚 | フード選び・食事のリズム | 分子栄養学・南イリノイ大学 |
| 2 | **くつろぎの巣** | 🏠 | 環境・安全基地 | ストレスケア・行動学 |
| 3 | **わくわく時間** | 🌿 | 散歩・遊び・嗅覚刺激 | エンリッチメント知識 |
| 4 | **体のサイン** | 💪 | ストレスサイン・予防ケア | 管理栄養士 × 予防栄養学 |
| 5 | **心のつながり** | 💞 | 飼い主マインド → 犬への影響 | **核心：コルチゾール同期研究** |

> [!IMPORTANT]
> **設計上の最重要ポイント**: 1〜4は「犬のセルフケア知識」で解決できるが、5だけは**飼い主自身が変わらないと解決しない**。
> 
> これが飯村さんの「犬だけ見てもダメ。飼い主が変わらなきゃ」と同じメッセージだが、**科学で裏付けている**のがきいちゃんの強み。

## 3. 診断の10問（きいちゃんの専門性で設計）

### 🍚 安心ごはん

**Q1. 愛犬の最後のごはんから翌朝のごはんまで、何時間あいていますか？**
- A. 6時間未満 → 1点
- B. 6〜8時間 → 2点  
- C. 8〜10時間 → 3点
- D. 考えたことがなかった → 0点

> 💡 **きいちゃんの専門知ポイント**: 夜間の空腹時間はMMC（消化管移行性運動複合体）が腸の「お掃除」をする時間。8〜10時間空けないと腸の修復が追いつかず、黄色い液を吐く原因になることも。**これ、獣医さんは逆に「こまめにあげて」って言うんですよね。そこが分子栄養学との差。**

**Q2. フードを選ぶとき、何を見ていますか？**
- A. SNSの口コミやランキング → 1点
- B. パッケージの雰囲気と値段 → 0点
- C. 原材料の上位5つと栄養成分 → 3点
- D. 獣医さんにすすめられたものをそのまま → 2点

**Q5. お散歩中、愛犬が匂いを嗅ぎたがったらどうしますか？**

- C. 衛生面が気になるのですぐ引く → 1点
- D. リードで引いて先に進ませる → 0点

> 💡 **きいちゃんの専門知ポイント**: セッションで実際に話していた内容。「しっかりワンちゃんに匂いを嗅がせてますか？引っ張ってないですか？行き先はワンちゃんに決めてもらってますか？」— これ、そのまま使える。

### 💞 心のつながり（★ 核心）

**Q9. 愛犬が体調を崩した時、あなたの最初の反応は？**
- A. 心配で何も手につかない → 0点
- B. まず落ち着いて症状を観察する → 3点
- C. すぐにネットで調べまくる → 1点
- D. すぐに病院に電話する → 2点

> 💡 **きいちゃんの原体験と完全一致**: 愛犬が生死の危機の時、「心配で離れられなかった」→「この子の人生だ」と切り離した瞬間に犬が回復。**この設問の意図そのもの。**

**Q10. あなたが不安やイライラを感じている時、愛犬の様子は？**
- A. 愛犬も落ち着きがなくなる → 0点（気づけている＝伸びしろ）
- B. 特に変わらないと思う → 1点（気づけていない可能性）
- C. そういえば、連動している気がする → 2点
- D. 自分が穏やかな時は愛犬も穏やかだと実感している → 3点

> 💡 **コルチゾール同期研究の体感版**: スウェーデン・リンショーピン大学（2019）の研究がまさにこれを証明。**飼い主のストレスが犬に伝わる。飯村さんは「感覚」で言っている。きいちゃんは「論文」で言える。**

## 8. まとめ — なぜこの設計が最強か

```
飯村さん:  インスタ → LINE → 40分説明動画 → 電話 → 面談 → 講座
                              ↑
                        ここが重い。たどり着く前に脱落

きいちゃん: SNS → 診断(3分) → LINE → 特典 → 教育動画 → 個別相談 → 講座
                   ↑              ↑
              軽い！楽しい    すぐ価値を受け取れる
```

| ポイント | なぜ強いか |
|---------|----------|
| **入口が診断** | 飯村さんはいきなり40分動画。きいちゃんは3分の診断で軽く入れる |
| **レーダーチャート** | 視覚的でシェアしたくなる → SNS拡散 |
| **不足栄養素に合わせた特典** | 「あなた専用の処方箋」感がある → LINE登録率アップ |
| **科学的根拠** | 飯村さんが「感覚」で語ることを「論文」で裏付け → 信頼性 |
| **栄養 + マインドの2本柱** | 飯村さんにはできない。きいちゃんだけの強み |
| **5つの栄養素フレームワーク** | 管理栄養士だからこそ語れる → ブランドの記号になる |
