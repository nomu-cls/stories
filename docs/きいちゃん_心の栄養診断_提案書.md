# 🐶 「あなたのワンちゃん、心の栄養足りてる？」診断 — 提案書

> **ポジション**: 既存の「犬のきもちクイズ（ライト）」と「飼い主マインド診断（ディープ）」の**間**に位置する、**中間層の診断コンテンツ**
> 
> **コンセプト**: ワンちゃんの行動を観察するだけでなく、**飼い主自身の関わり方が犬の「心の栄養」に直結している**ことに気づかせる橋渡し

---

## 1. なぜ「心の栄養」なのか

### きいちゃんの核心テーゼとの接続

きいちゃんのビジネスの核心は「**飼い主のマインドが犬の健康に直結する**」こと。

既存の2つの診断は：
- **犬のきもちクイズ** → 犬の知識（＝体の栄養みたいなもの）
- **飼い主マインド診断** → 飼い主の内面パターン（＝やや深い・重い）

この間に「**心の栄養**」という概念を挟むことで、自然な橋渡しが生まれる。

| | 犬のきもちクイズ | **心の栄養診断（NEW）** | 飼い主マインド診断 |
|---|---|---|---|
| **深さ** | ライト | ミドル | ディープ |
| **主語** | 犬のこと | 犬 ← → 飼い主 | 飼い主のこと |
| **感情** | 「へぇ〜！」 | 「うちの子、大丈夫かな…？」 | 「ハッ…自分のことだ」 |
| **行動** | シェアしたくなる | もっと知りたい | LINE登録して学びたい |

### 科学的裏付け

> [!NOTE]
> **スウェーデン・リンショーピン大学の研究（2019年, Scientific Reports掲載）**
> 
> 58組の犬と飼い主の毛髪コルチゾール（ストレスホルモン）を分析した結果、**飼い主のストレスレベルが犬のストレスレベルと長期的に同期**していることが判明。しかも、影響の方向は「飼い主 → 犬」であり、犬の性格ではなく**飼い主の性格特性**が犬のコルチゾール値を左右していた。

→ つまり「ワンちゃんの心の栄養が足りているか？」は、**飼い主自身の状態を映す鏡**になる。

## 2. 診断の設計

### 2-1. 全体像

```
「あなたのワンちゃん、心の栄養足りてる？」診断
    ↓
  10問のチェック式（2〜3分）
    ↓
  5つの「心の栄養素」ごとにスコア表示
    ↓
  レーダーチャートで可視化
    ↓
  不足している栄養素 → 具体的なケア方法（＋きいちゃんの専門知識）
    ↓
  「もっと深く知りたい？」→ 飼い主マインド診断 or LINE登録
```

### 2-2. 5つの「心の栄養素」

動物福祉の国際基準「5つの自由」をベースに、きいちゃんの専門性と掛け合わせたオリジナルフレームワーク：

| # | 心の栄養素 | 元になる概念 | きいちゃんの専門領域との接続 |
|---|-----------|------------|-------------------------|
| 🟡 | **安心ごはん** | 飢え・渇きからの自由 | **分子栄養学** — フード選びの基準、空腹時間の設計 |
| 🟢 | **くつろぎスペース** | 不快からの自由 | ストレスケア — 照明・音・安全基地の確保 |
| 🔵 | **ワクワク体験** | 本来の行動がとれる自由 | メンタルエンリッチメント — 嗅覚散歩、知育トイ |
| 🟣 | **安心のきずな** | 恐怖・抑圧からの自由 | **飼い主マインド** — コルチゾール同期、感情伝播 |
| 🔴 | **からだのケア** | 痛み・病気からの自由 | **栄養管理** — 予防栄養学、腸内環境 |

> [!IMPORTANT]
> **ポイント**: 5つの栄養素のうち「🟣 安心のきずな」だけが**飼い主マインドに直結**。
> ここのスコアが低い人 → 「あなたの心の状態がワンちゃんに影響しているかも？」→ 飼い主マインド診断へ誘導。

#### 🟡 安心ごはん（栄養）

**Q1. 愛犬の夜ご飯と翌朝ご飯の間隔は？**

| 選択肢 | スコア |
|--------|--------|
| A. 6時間未満（寝る直前にあげている） | 1点 |
| B. 6〜8時間くらい | 2点 |
| C. 8〜10時間 | 3点 |
| D. 考えたことがなかった | 0点 |

> 🔬 **解説**: 夜間の空腹時間（8〜10時間）は腸の修復「MMC（消化管移行性運動複合体）」が働くために必要。寝る直前の食事は腸の回復を妨げます。

## 6. 競合との差別化

### 6-1. 市場にある類似コンテンツ

| 既存コンテンツ | 特徴 | きいちゃんとの違い |
|--------------|------|-----------------|
| ペット保険会社のストレスチェック | 行動面のチェックリストのみ。一般的 | **飼い主マインドへの橋渡しがない** |
| 獣医監修の健康チェック | 身体的症状中心。病院への受診促進 | **心理面・栄養面の統合がない** |
| アニマルコミュニケーター系 | スピリチュアル寄り。科学的根拠なし | **分子栄養学 × コルチゾール研究で差別化** |

### 6-2. きいちゃんだけの差別化ポイント

1. **「5つの自由」× 分子栄養学** = 動物福祉の国際基準をベースにしつつ、栄養学の専門知識で深みを出す
2. **飼い主マインド診断との接続** = 「犬の状態」を入口に「飼い主の内面」に気づかせる設計は他にない
3. **レーダーチャート** = 「何が足りないか」が視覚的に分かる → シェアしたくなる → 拡散

## 7. 実装メモ

### 技術的な選択肢

| 方式 | メリット | デメリット |
|------|---------|----------|
| **UTAGEのフォーム** | LINEと直接連携、タグ付け自動化 | レーダーチャート表示が難しい |

> [!TIP]
> **推奨**: 独自ページ（HTML/JS）で作成 → 結果画面からLINE登録ボタンへ誘導。
