5 TYPES

🐾 5つの飼い主タイプ

🫧
TYPE A

心配バリア型

愛犬のことが常に心配。ちょっとした体調変化で不安になり、獣医さんに駆け込んでしまう。「この子に何かあったら…」が頭から離れない。
🔬 犬への科学的影響：飼い主のストレスホルモン（コルチゾール）は犬に伝播。過度な心配が犬の免疫力を下げ、食べムラや消化不良の原因に。
→ マインドセット講座で「見守る力」を学ぶ
🌀
TYPE B

情報ぐるぐる型

SNSやネットで情報を集めすぎて、何が正しいか分からなくなる。フードを頻繁に変えてしまう。「グレインフリーがいい」「いや手作りが」と右往左往。
🔬 犬への科学的影響：フードの頻繁な変更は腸内細菌叢を不安定にする。飼い主の迷いは犬の食事への不安感にもつながる。
→ 栄養講座で「選ぶ基準」を持つ
😢
TYPE C

ぜんぶ私のせい型

愛犬の問題行動や不調を「すべて自分のせい」と思い込む。罪悪感でいっぱいになり、過干渉になったり逆に何もできなくなったりする。
🔬 犬への科学的影響：飼い主の罪悪感→過干渉は、犬の自律神経バランスを乱す。結果的に嘔吐・下痢・カイカイなどの身体症状として現れることも。
→ マインドセット講座で「切り離す力」を学ぶ
🫂
TYPE D

離れられない型

愛犬と一瞬たりとも離れたくない。留守番させることに強い罪悪感。仕事も休みがちに。自分の時間や趣味がなくなっていく。
🔬 犬への科学的影響：共依存状態は犬の分離不安を強化する。飼い主が離れると吠え・破壊行動・自傷行為の原因に。犬の「自立する力」を奪ってしまう。
→ マインドセット講座で「飼い主軸」を育てる
📏
TYPE E

完璧を目指す型

栄養計算を厳密にやり、「これで足りてるのか」を常に気にする。少しでもバランスが崩れると不安になる。手作り食のハードルが自分の中で高くなる。
🔬 犬への科学的影響：飼い主の緊張感は食事の場に伝わり、犬が食事を楽しめなくなる。結果的に消化吸収効率も低下。
→ 栄養講座で「柔軟な基準」を身につける
ポイント：5タイプ全てが、きいちゃんの栄養講座（B・E型）とマインドセット講座（A・C・D型）で解決できる設計
QUESTIONS

📝 診断質問（7問）

  • Q1

    愛犬の食欲がない日、最初に何をしますか？

    測定：不安度 vs 観察力（心配バリア型 ↔ 冷静観察型）

  • Q2

    ネットで「この食材は犬に危険」と見たらどうしますか？

    測定：情報依存度（情報ぐるぐる型の判別）

  • Q3

    愛犬を一人で留守番させるとき、何を感じますか？

    測定：分離不安の投影度（離れられない型の判別）

  • Q4

    獣医さんの診断に納得がいかない時、どうしますか？

    測定：自己判断力 vs 権威依存度

  • Q5

    この1年で、愛犬のフードを何回変えましたか？

    測定：情報振り回され度（情報ぐるぐる型の数値判定）

  • Q6

    愛犬が体調を崩した時、「自分のせいかも」と思いますか？

    測定：罪悪感度（ぜんぶ私のせい型の判別）

  • Q7

    愛犬の手作りごはん、栄養計算はどのくらいしますか？

    測定：完璧主義度（完璧を目指す型の判別）
